# portfolio

## Estructura

- `index.html`: markup principal de la landing.
- `assets/css/styles.css`: estilos personalizados compartidos.
- `assets/js/tailwind.config.js`: configuración de Tailwind centralizada.

## Objetivo

El proyecto mantiene el mismo diseño y contenido, pero con una separación más limpia de responsabilidades para facilitar mantenimiento y futuras mejoras.
